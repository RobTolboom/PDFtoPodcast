ROLE

INPUTS

- (optional) PDF_CONTENT

OUTPUT FORMAT (STRICT)
Return ONLY JSON conforming to appraisal_v1 with:

- Include: tools, amstar2, robis, synthesis_quality, grade_per_outcome, bottom_line, required_followup, (optional) generator

- grade_per_outcome:
  - Use the review’s own certainty judgments if provided; otherwise infer cautiously from reported RoB, inconsistency, indirectness, imprecision, publication bias.
